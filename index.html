기술 영역

SKILLS

  • H5 HTML

    웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 하고 있습니다. 웹 표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 이용해 전체 레이아웃을 설계합니다. 또한 웹 접근성 준수를 위해 키보드에 대한 대응과 적절한 대체 텍스트를 제공하고자 노력합니다.

  • C3 CSS

    크로스브라우징을 위해 안정적인 CSS 속성을 사용해서 HTML Element를 배치하고 디자인합니다. 레이아웃 제작에서 기존 방식인 float과 최신 트랜드인 flex 기법을 적재적소에 활용할 수 있습니다. 또한, 생산성과 협업을 위하여 CSS 전처리기인 SCSS 기본 문법도 숙지하고 있습니다.

  • JS JQUERY

    이미지 슬라이드, 마우스 휠에 반응하는 슬라이드 배너, 내비게이션, 탭 메뉴, 아코디언 등 웹 퍼블리싱에 필요한 제이쿼리는 플러그인을 사용하지 않고 순수 제이쿼리 코딩으로 제작할 수 있습니다. 또한 필수적인 자바스크립트 활용은 가능하며 주로 제이쿼리와 병행해서 활용합니다.

  • RW RESPONSIVE WEB

    반응형 웹 제작에 있어서 크로스브라우징을 위해 안정적인 기존방식인 float, position, Media Query를 활용한 반응형 웹 제작 경험을 충분히 가지고 있습니다. 모바일 웹 앱 제작의 경우에는 flex를 활용합니다.

  • GIT GIT

    공동 개발에서 효율적인 버전관리를 위해 git 명령어를 학습하여 브랜치 사용이 가능합니다.

  • PS PHOTOSHOP

    웹 퍼블리셔 직군 특성에 맞게 Adobe Photoshop 기본 툴을 다루는 것이 가능합니다.

프로젝트 개요

페이지마다 다른 테마로 구성된 제주맥주 웹사이트에 매력을 느껴 모든 서브 페이지를 제작하였습니다. 모바일 내비게이션은 사용자가 좀 더 직관적으로 알아볼 수 있도록 약간의 리 디자인을 진행하였습니다. floating action button에서 문의하기 모달창은 제주맥주 대표색채로 재작업하였으며, 키보드 사용자를 위해 키보드 접근성도 고려하여 제작하였습니다. 또한 SNS 연동까지 완료함으로써 실제 웹사이트처럼 원하는 정보를 획득할 수 있습니다.

프로젝트 개요

꼬띠샵은 2021 GDWEB 디자인 어워즈 WEB 부문에서 WINNER PRIZE를 수상하였습니다. 프리미엄 잡지 컨셉으로 위트있는 아이콘들과 일러스트 비주얼이 매력적이어서 클론코딩 대상으로 삼았습니다. 브랜드의 개성을 유지하면서도 꼬띠샵만의 캐릭터를 활용하여 마우스 호버 효과, 탑 버튼, 마우스 커서 모양 변경 등 다양한 요소를 곳곳에 배치하여 재미있습니다. 사용자의 UI 부분을 고려하여 모든 서브 페이지를 제작하였으며, 페이지 곳곳의 숨은 애니메이션은 주로 keyframes로 구현하였습니다.
